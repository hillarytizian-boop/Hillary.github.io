# Tizian Dashboard

This is a fully functional **Tizian social media dashboard** with:

- Post scheduler (TikTok/WhatsApp)
- Simulated auto-posting
- AI caption generator
- Analytics chart
- Best posting time analyzer
- Admin panel

## How to Upload

1. Create a GitHub repository.
2. Upload all 6 files from your phone.
3. Open `index.html` in a browser to use the dashboard.
4. Open `admin.html` for admin panel.

## Notes

- Simulated TikTok/WhatsApp posting logs to console.
- Posts are stored in browser `localStorage`.
- Works fully offline.
